Detailed Description

Enumerado que define los campos de una especificación de ejercicio.

Este enumerado se utiliza para abstraer el acceso a los atributos de una clase ExerciseEspec, permitiendo validar, editar o mostrar dinámicamente los campos asociados a un ejercicio, como el número de series, repeticiones, duración o grupo muscular objetivo.
