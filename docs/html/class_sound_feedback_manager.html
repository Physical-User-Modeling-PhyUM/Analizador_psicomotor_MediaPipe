Detailed Description

Clase encargada de reproducir sonidos asociados a condiciones específicas del ejercicio.

Esta clase permite la asociación entre ConditionType y archivos de sonido. Soporta reproducción de múltiples sonidos simultáneos, control de volumen, y configuración de filtros según el nivel de criticidad (informativo, alerta, crítico).
