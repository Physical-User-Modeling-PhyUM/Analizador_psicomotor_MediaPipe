Detailed Description

Enumerado que define los niveles de movilidad física de un usuario.

Este archivo proporciona un enum MobilityLevel para representar la capacidad de movilidad en perfiles de usuario, junto con funciones auxiliares para su conversión desde y hacia cadenas de texto.
