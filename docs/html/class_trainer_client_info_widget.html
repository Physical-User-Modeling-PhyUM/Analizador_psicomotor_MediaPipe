Detailed Description

Widget que permite al entrenador visualizar y modificar los perfiles detallados de sus clientes.

Cada perfil incluye información general, funcional, morfológica, médica, de estilo de vida y rango articular (ROM). Las tablas permiten edición directa, así como añadir o eliminar registros.
