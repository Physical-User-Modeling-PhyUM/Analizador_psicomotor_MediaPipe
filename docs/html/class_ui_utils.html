Detailed Description

Clase de utilidades para convertir datos entre estructuras internas y modelos de tabla en la UI.

Esta clase sirve como puente entre el modelo de datos y la interfaz de usuario. Facilita la carga de datos en tablas (UiTableModel) y la conversión de los datos modificados por el usuario de vuelta a las estructuras originales. También permite mostrar filtros, poblar tablas con enums, y gestionar transiciones de estados en ejercicios.
