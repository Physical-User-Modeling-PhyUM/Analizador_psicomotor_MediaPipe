Detailed Description

Clase encargada de coordinar el acceso a entrenamientos y sesiones mediante un repositorio.

Esta clase encapsula la lógica de acceso a los datos de entrenamiento (workouts), ejercicios, sesiones activas, y resúmenes asociados. Es el punto de entrada para crear, editar, listar y ejecutar sesiones de entrenamiento.
