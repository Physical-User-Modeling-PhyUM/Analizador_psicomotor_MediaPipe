Detailed Description

Clase que encapsula todos los registros físicos, médicos y de estilo de vida de un cliente.

Organiza los datos por categorías (general, funcional, morfológica, médica, ROM, estilo de vida y adicional), y proporciona métodos seguros para su acceso y validación con límites definidos por tipo de campo.
