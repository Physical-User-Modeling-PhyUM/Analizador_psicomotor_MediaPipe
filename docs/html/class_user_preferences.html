Detailed Description

Clase que almacena las preferencias de configuración de un usuario.

Esta clase permite gestionar configuraciones como el modo dual de cámara, el uso de sonidos informativos y de alerta, el estado de silencio general, y una estructura JSON que representa conexiones entre keypoints. La configuración puede exportarse/importarse desde una cadena JSON.

QHash< QPair< int, int >, QString > UserPreferences::getConnections ( ) const

Devuelve el conjunto de conexiones entre keypoints.

Convierte la representación JSON de conexiones a una estructura QHash.

Returns
Hash con pares de enteros como clave y etiquetas descriptivas como valor.

Cada conexión se espera como un array de dos enteros en la forma [from, to]. Se genera una etiqueta tipo "from-to" como valor para cada par.

Returns
