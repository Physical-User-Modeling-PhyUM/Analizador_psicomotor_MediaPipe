Detailed Description

Cabecera centralizada que incluye todos los enumerados utilizados en el sistema.

Este archivo agrupa todas las definiciones de enumerados relacionados con los usuarios, preferencias, tipos de feedback, roles, perfiles y campos de base de datos. Facilita su inclusión en cualquier parte de la aplicación sin necesidad de importar cada enumerado por separado.

Enumerados incluidos:

  • UserFieldEnum
  • FeedbackModeEnum
  • GenderEnum
  • MobilityLevelEnum
  • PreferencesFieldEnum
  • UserTypeEnum
  • TypeBDEnum
  • ProfileEnums (ROM, Lifestyle, etc.)
