Detailed Description

Clase que se encarga de la validación de entradas en formularios del sistema.

Utiliza reglas predefinidas para asegurar que los valores ingresados por el usuario cumplan con los formatos requeridos (email válido, campos no vacíos, contraseñas seguras, números positivos, etc.). En caso de error, se emite una señal para que la interfaz pueda notificar al usuario.
