Detailed Description

Declaración de la clase UiTableModel, modelo de tabla para representar y editar datos en la interfaz gráfica.

Esta clase extiende QAbstractTableModel y proporciona un modelo de datos bidimensional utilizado principalmente en componentes de tipo QTableView. Permite cargar, editar y validar datos estructurados por filas y columnas. Incorpora mecanismos para marcar columnas como de solo lectura y conectar un gestor de validación externo para comprobar las modificaciones realizadas por el usuario.
