Detailed Description

Representa una restricción angular para un estado específico de un ejercicio.

Define los límites y condiciones que deben cumplirse durante un movimiento para considerarse válido. Estas restricciones son utilizadas por el sistema de análisis postural para generar retroalimentación en tiempo real.
