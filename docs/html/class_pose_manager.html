Detailed Description

Controlador de alto nivel que captura y analiza poses del usuario en tiempo real.

Coordina la ejecución de scripts Python para capturar datos de cámaras, lee keypoints desde memoria compartida, crea objetos Pose, y los analiza usando una máquina de estados para generar feedback durante el ejercicio.
