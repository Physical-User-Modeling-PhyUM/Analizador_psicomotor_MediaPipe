Detailed Description

Clase que representa una sesión de entrenamiento ejecutada por un cliente.

Contiene los datos esenciales como identificador, cliente, fecha, resumen del ejercicio realizado, duración y el informe detallado generado por el sistema de análisis de movimiento (SesionReport).
