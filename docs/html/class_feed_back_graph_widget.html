Detailed Description

Declaración de la clase FeedBackGraphWidget para la visualización gráfica del feedback de condiciones.

Widget gráfico que representa visualmente el resumen de condiciones de un ejercicio.

Muestra una gráfica circular tipo donut con porcentajes de condiciones óptimas, de alerta y críticas. Esta visualización permite una rápida comprensión del estado general del rendimiento.
