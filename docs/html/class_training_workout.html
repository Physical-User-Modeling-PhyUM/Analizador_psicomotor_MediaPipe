Detailed Description

Clase que encapsula un plan de entrenamiento con múltiples ejercicios.

Permite organizar ejercicios específicos (ExerciseEspec) dentro de un entrenamiento, manteniendo atributos como nombre, descripción, autor, duración estimada y funcionalidades para modificar la lista de ejercicios.
