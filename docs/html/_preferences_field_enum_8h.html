Detailed Description

Enumerado que representa los campos configurables en las preferencias del usuario.

Define las claves de configuración relacionadas con la personalización de la experiencia del usuario, tales como idioma, feedback visual y auditivo. Incluye funciones auxiliares para conversión entre el enum y su representación textual.
