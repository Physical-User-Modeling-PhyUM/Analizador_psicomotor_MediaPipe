Detailed Description

Clase que representa a un preparador físico dentro del sistema.

Esta clase extiende la funcionalidad de User para incorporar información profesional específica de los entrenadores, como sus calificaciones, especialidad, experiencia y currículum.

Constructor que inicializa un entrenador con los datos proporcionados.

Constructor de FitnessTrainer.

Parameters
dataMapMapa de campos clave-valor que contienen la información del entrenador.
parentObjeto padre dentro de la jerarquía de Qt.

Este constructor valida que los campos obligatorios del usuario estén presentes en el dataMap. Si la validación es satisfactoria, se procede a establecer los datos básicos del usuario y los atributos específicos del entrenador.

Además, se carga la imagen de perfil si está disponible y se aplica una validación adicional sobre los años de experiencia (0 a 80 años).

Parameters
dataMapMapa con datos clave del entrenador, incluyendo nombre, correo, contraseña, etc.
parentObjeto padre (por defecto nullptr).
