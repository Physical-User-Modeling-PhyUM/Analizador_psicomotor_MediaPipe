Detailed Description

Repositorio responsable de la gestión de datos de entrenamiento en la base de datos.

Encapsula las operaciones CRUD para objetos como ExerciseEspec, TrainingWorkout y TrainingSesion, además de funciones auxiliares para listar elementos y convertir datos a estructuras de persistencia (DBTable).
