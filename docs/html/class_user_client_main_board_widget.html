Detailed Description

Declaración de la clase UserClientMainBoardWidget, que gestiona la vista principal del cliente en la aplicación.

Widget principal del cliente, que actúa como panel de navegación entre secciones.

Este widget contiene el stackedWidget con las vistas:

  • Información del cliente
  • Selección de sesiones de entrenamiento
  • Perfil del cliente
  • Configuración personal
