Detailed Description

Representación en memoria de una tabla con columnas y filas tipo QVariant.

Proporciona métodos para insertar y acceder a datos tabulares de forma estructurada. Útil para gestionar datos provenientes de SQLite u otros motores relacionales en la lógica de la aplicación.
