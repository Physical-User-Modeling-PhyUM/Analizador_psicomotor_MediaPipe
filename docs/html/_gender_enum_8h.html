Detailed Description

Enumerado que representa el género de un usuario.

Este archivo define un enum Gender con valores posibles para el género biológico, así como funciones auxiliares para convertir entre Gender y QString. Se utiliza principalmente en perfiles de usuario o formularios relacionados.
