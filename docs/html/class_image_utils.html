Detailed Description

Utilidades estáticas para la comparación de imágenes en formato OpenCV.

Esta clase contiene métodos que permiten realizar operaciones comunes de comparación entre imágenes, como verificar si dos imágenes son iguales pixel a pixel tras una posible normalización de canal de color.

Compara dos imágenes y determina si son visualmente iguales.

Compara dos imágenes para verificar si son visualmente iguales.

Este método compara el tamaño, tipo y contenido de las dos imágenes. Si ambas imágenes tienen tres canales (formato BGR), se convierten previamente a escala de grises para una comparación neutral. Luego se calcula la diferencia absoluta y se comprueba si existe algún pixel diferente.

Parameters
mat1Primera imagen a comparar (cv::Mat).
mat2Segunda imagen a comparar (cv::Mat).
Returns
true si las imágenes son idénticas visualmente; false en caso contrario.

Este método realiza una comparación pixel a pixel entre las dos imágenes.

  • Verifica primero que ambas imágenes tengan el mismo tamaño y tipo.
  • Si ambas son imágenes en color (3 canales), se convierten a escala de grises para asegurar una comparación neutral.
  • Se calcula la diferencia absoluta entre las imágenes y se cuenta el número de píxeles diferentes.

mat1Primera imagen.

Returns
true si no hay diferencias entre las imágenes; false en caso contrario.
