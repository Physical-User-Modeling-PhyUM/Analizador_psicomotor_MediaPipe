Detailed Description

Declaración de la clase UserClientinfoWidget, encargada de mostrar los datos personales del cliente.

Widget que muestra la información del perfil de un usuario cliente.

Incluye datos personales, plan asignado, fechas de conexión y una imagen de perfil editable.
