Detailed Description

Widget principal que permite al entrenador gestionar clientes, métricas y entrenamientos.

Contiene varias secciones organizadas en un QStackedWidget accesibles mediante un menú lateral. Gestiona además un buffer de clientes en memoria para optimizar el acceso.
