Detailed Description

Clase responsable de la autenticación y gestión de contraseñas de los usuarios del sistema.

Permite autenticar usuarios por nombre de usuario o correo electrónico, validar contraseñas usando hash SHA-256, registrar nuevos clientes con contraseña cifrada y permitir el cambio seguro de contraseña. Emite señales que permiten notificar el resultado del proceso de autenticación o registro.
