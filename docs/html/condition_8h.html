Detailed Description

Define el tipo de datos Condition utilizado para representar eventos biomecánicos detectados durante el ejercicio.

Este archivo incluye la definición del enumerado ConditionType, la estructura Condition y funciones auxiliares para conversión y comparación. Representa eventos como sobrecarga, desviación de simetría, tiempos límite superados, etc. Utilizado para generar feedback al usuario en tiempo real.
